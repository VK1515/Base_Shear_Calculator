## About

This tool implements seismic base shear calculations as per **IS 1893 (1962–2025)** and is intended for academic and professional reference use.

It supports:
- 1962, 1966, 1970, 1984, 2002, 2025 codes
- Unit-safe input (W in kN, results in kN)
- Export to Excel and PDF
- Graphical base shear distribution

## Usage

Run locally:
```bash
pip install -r requirements.txt
streamlit run seismic_calculator.py
```
